## How to Test the App

### 1. Authentication
- App uses Replit Auth (secure OAuth)
- Click "Sign In" on landing page
- Use provided test credentials above
- Or create new account with any email

### 2. Core Features to Test

**Feed View:**
- Browse miracle testimonies and faith posts
- Tap "Pray" button (cross icon) to pray for posts
- Tap "Love" button (heart) to show support
- Tap "Comment" to add encouraging messages

**Create Content:**
- Tap blue "+" button to create new post
- Share your own miracle or faith experience
- Optional: Add photo and location
- Post appears in community feed

**Social Features:**
- View user profiles by tapping names
- Send connection requests to other believers
- Private messaging between connected users

### 3. Navigation
- **Feed:** Main community posts
- **Discover:** Browse by location and themes
- **Messages:** Direct conversations
- **Profile:** Personal settings and posts

### 4. Key User Flows

**New User Journey:**
1. Open app → See landing page
2. Tap "Sign In" → Authenticate
3. Complete profile setup
4. Browse community feed
5. Create first post sharing faith experience

**Community Interaction:**
1. Find inspiring post
2. Tap "Pray" to support
3. Add encouraging comment
4. Connect with poster
5. Continue conversation in messages

### 5. Technical Features
- **Offline Support:** Works without internet
- **PWA Install:** Can install as native app
- **Location Services:** Optional location sharing
- **Image Upload:** Photo sharing capability

### 6. Content Guidelines
- Faith-based content only
- Positive, encouraging testimonies
- Family-friendly for all ages
- Community moderation available

### 7. Special Notes for Reviewers
- App targets Christian community
- Focuses on miracle testimonies and faith sharing
- No paid features or subscriptions
- COPPA compliant for children 13+
- Privacy policy at: https://proofofamiracle.com/privacy

### 8. Expected User Experience
Users will find:

- Safe space for sharing testimonies
- Tools for spiritual connection
