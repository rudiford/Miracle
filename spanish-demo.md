# Spanish Language Translation Demo

## How it works:
1. **Language Selector**: Top-right corner shows EN/ES dropdown
2. **Instant Translation**: Click "ES" to see immediate Spanish translation
3. **Persistent Choice**: Selection saved to localStorage
